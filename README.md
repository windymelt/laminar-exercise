# Bouncing DVD Logo in Laminar

## Prerequisites

- scala development environment (sbt)
- Node.js (npm)

## Run

To run:

```sh
$ npm i
$ npm run dev
```

## What is Laminar

[Laminar](https://laminar.dev/) is UI library for [Scala.js](https://www.scala-js.org/).

This application is built with Scala 3, Laminar, Scala.js, Vite.
